

# Introduction #

This guide helps users updating the eConference tool without having to manually
download and uncompress a new archive, each and every time a new build is release.
The update is performed using the built-in, auto-update feature available via Eclipse RCP.


# Details #
Select the `Help > Install New Software` menu entry. Then, add a new site entering a description and
the following url: `http://econference4.googlecode.com/svn/update/`, as
described in **Figure 1**.
